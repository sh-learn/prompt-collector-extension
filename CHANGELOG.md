# 更新日志

## 0.2.0 - 2026-05-23

- 新增 X/Twitter GraphQL 响应观察能力，用于捕获视频 MP4 直链和长帖/长评论完整正文。
- 新增视频候选素材，和提示词、图片候选一起勾选；选中的视频会上传到飞书附件字段。
- 新增 X 视频封面图兜底：未捕获到视频文件直链时，将封面图作为图片候选保留。
- 优化 X 主帖和已加载评论的候选提取，并支持将多个已选候选合并保存到同一条飞书记录。
- 新增 Seedance / 视频提示词和分镜 / 故事板提示词识别，并收紧 marker 解析，避免误判“故事板提示词，拆解在评论区”这类正文。
- 优化飞书字段自动填充：字段名包含 `Seedance` / `视频提示词` 或 `分镜` / `故事板` 时，优先填入对应类型提示词。
- 优化候选区交互：勾选或取消勾选时只刷新已选字段和素材预览，不再自动跳回顶部。
- 补充 X 采集限制和测试注意事项说明。

## 0.1.0 - 2026-05-21

- 初始 Chrome MV3 插件版本，支持采集提示词、图片、来源链接和元数据并保存到飞书多维表格。
- 新增飞书 OAuth、本地保存 App Secret、绑定 Base、选择数据表、读取字段结构和附件上传。
- 新增页面内侧边采集面板，支持字段预览编辑和保存到飞书。
